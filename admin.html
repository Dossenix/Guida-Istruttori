Solo GitHub

Pubblica un aggiornamento

Il modulo prepara una richiesta GitHub. Dopo l'invio, una GitHub Action aggiunge il testo sia al registro aggiornamenti sia alla sezione scelta del regolamento.

Apri GitHub

Richiesta che verrà inviata a GitHub

Compila il modulo per preparare l'invio.

Dopo aver aperto GitHub, controlla il testo e premi "Submit new issue". Se il testo non appare già compilato, usa "Copia richiesta" e incollalo nel corpo della issue. Solo le richieste inviate dall'account Dossenix vengono pubblicate.
